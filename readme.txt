== Description ==
Budhilaw Blog is a simple and clean WordPress blog theme.
It is designed and developed primarily for personal blogs.
It is fully responsive and works well on all devices.
It is compatible with the Gutenberg editor and supports the dark mode feature.
It is also translation ready and can be easily translated into multiple languages.
